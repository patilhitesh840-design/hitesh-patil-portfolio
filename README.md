# AI-Powered 3D Portfolio

An interactive, premium 3D portfolio website highlighting skills, education, and including an interactive AI agent. The visual design utilizes dark space aesthetics with electric cyan and purple glows combined with modern glassmorphism.

## Features

- **Interactive 3D Space Background**: A responsive starfield built with Three.js that tilts on mouse movement and speeds up on scroll.
- **3D Skills Sphere**: An interactive sphere composed of tech skills drifting in a 3D coordinate orbit. Drag to rotate and interact.
- **AI introduction**: Typewriter boot terminal introducing Hitesh upon entering the portfolio.
- **Interactive AI Chatbot Console**: A floating chatbot widget to answer questions about skills, education, and contact channels.
- **Education Timeline**: A stylish vertical timeline illustrating milestones.
- **Contact Form**: Glassmorphic form with validation and submit feedback.
- **Responsive Layout**: Works seamlessly across mobile, tablet, and desktop viewports.

## Tech Stack

- **Three.js** (WebGL 3D rendering)
- **GSAP** (Scroll & entrance animations)
- **Remix Icons** (Modern UI icons)
- **Vanilla CSS** (Custom styling & responsive grid layouts)
- **Vanilla JavaScript** (State logic & interactive events)

## Quick Start

1. Double-click or open **`index.html`** in any web browser.
2. No local node dependencies or servers are required!
